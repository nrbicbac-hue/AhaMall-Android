# 아하몰 Aha mall 안드로이드 앱

[아하몰 APK 다운로드](https://github.com/nrbicbac-hue/AhaMall-Android/releases/download/v1.0.0/AhaMall-1.0.0.apk)

쇼핑몰 `https://www.imall7.com/me2`를 안드로이드 앱으로 이용할 수 있도록 만든 WebView 기반 프로젝트입니다.

## 포함 기능

- 앱 이름: **아하몰**
- 쇼핑몰 홈 화면 자동 연결
- 로그인·장바구니 쿠키 유지
- 결제 앱과 외부 앱 연결(`intent://`, 전화, 문자, 이메일 등)
- 상품문의·회원가입용 사진/파일 첨부
- 파일 다운로드
- 앱 내 뒤로가기와 페이지 로딩 표시
- 인터넷 오류 화면 및 다시 시도
- 아하몰 전용 주황색 쇼핑백 아이콘

## 가장 쉬운 APK 만들기

1. GitHub의 **Actions** 탭에서 `Build Aha Mall APK` 실행 결과를 확인합니다.
2. 완료되면 위의 **아하몰 APK 다운로드** 링크에서 설치 파일을 받습니다.
3. APK를 휴대폰이나 다운로드 페이지에서 배포할 수 있습니다.

`main` 브랜치가 변경되면 GitHub Releases의 `v1.0.0` APK가 자동으로 갱신됩니다.

## Android Studio에서 실행

1. Android Studio에서 이 폴더를 엽니다.
2. Gradle 동기화가 끝나면 실제 휴대폰을 연결합니다.
3. Run을 눌러 주문, 결제, 로그인, 파일 첨부를 시험합니다.

## 출시 전 확인 사항

- 카드 결제와 간편결제 앱 전환
- 휴대폰 본인인증과 새 창
- 로그인 유지 및 장바구니
- 반품·교환 사진 첨부
- 개인정보처리방침 및 고객센터 표시

Google Play에 올릴 때는 소유자 명의의 정식 서명키로 `release` 앱을 만들어야 합니다. 현재 자동 빌드는 빠른 시험·직접 배포용 디버그 APK입니다.
